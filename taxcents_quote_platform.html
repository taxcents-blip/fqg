How to use the platform

      
1. Client → 2. Packages (add one or more) → 3. Quote (edit, export).

      
Quick start

      
    
        
  • Client — Enter client name and tax year, then click Continue.
    • 
        
  • Packages — Click a package (A–H). Add base only or go to Add-ons for optional forms. Bookkeeping and consulting: type them in "Additional line items" on the Add-ons screen. Click Generate quote when ready.
    • 
        
  • Quote — Review, edit line items, then export via Preview, PDF, or Excel.
    • 
      

      
How to use the platform

      
This tool produces professional fee quotes (addendums) that match the Taxcents PDF/Excel template. Only the client name, date of engagement, and the fee table change per quote; branding, terms, and payment details stay consistent. You can add rows, custom columns (e.g. Remarks), and edit descriptions, quantities, fees, and notes—PDF and Excel adapt to your table.


      
Step by step

      
Step 1 — Client details
Enter the client name or entity, date of engagement (e.g. DD/MM/YYYY), and the tax year (e.g. 2025 or FY2025). You can change these later from the quote summary. Click "Continue to package selection".

      
Step 2 — Packages
Click a package (A–H) to add it. Each package will ask for add-ons before being added. Your added packages appear in the box below; you can remove any one or clear all. When ready, click "Generate quote".

      
Step 2b–d — Confirm and add-ons
After selecting a package, you’ll confirm it, then see optional add-ons (e.g. Form 5471, Form 8938). Use "Additional line items" to type bookkeeping or consulting (or any form code) — we match to our services and set the fee. Use Add with base only to skip add-ons. For State Tax Filing in additional line items, you pick which U.S. state(s). When ready, click "Add to quote".

      
Step 3 — Quote summary
Review and edit the quote. Switch to Edit mode to change descriptions, quantities, fees, currency, and notes; add or remove rows; and add custom columns (up to 15). Use "Add another package" to return to Step 2, or export from here.


      
Exports

      
    
        
  • Preview — View the full addendum as it will appear in PDF/Excel.
    • 
        
  • PDF — Opens the print dialog; choose "Save as PDF" to download.
    • 
        
  • Excel — Downloads an Excel file with the fee table (Description, Qty, Rate, Currency, Notes, and any custom columns).
    • 
        
  • Email — Enter recipients and open your mail client with a pre-filled message; attach the PDF or Excel after downloading.
    • 
        
  • Save draft / Load draft — Saves or restores the current quote in this browser only.
    • 
      


      
Limits

      
Custom columns: up to 15 (so PDF/Excel stay readable). Client name: up to 200 characters. Column names: up to 50 characters. Tax year: 2000–2100.


      
Tips

      
    
        
  • Use the stepper (Step 1 / 2 / 3) at the top to jump back to Client details or Packages when the step is enabled.
    • 
        
  • In the quote, click "Done editing" when you finish changes so the view is clean before exporting.
    • 
        
  • Keyboard shortcuts in Edit mode: Ctrl+Z undo, Ctrl+Y redo, Ctrl+S save draft. Click "⌨ Shortcuts" in the header for the full list.
    •
